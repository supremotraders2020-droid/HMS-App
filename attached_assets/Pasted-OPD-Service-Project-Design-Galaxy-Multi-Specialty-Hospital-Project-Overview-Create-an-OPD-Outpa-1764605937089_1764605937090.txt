OPD Service Project Design - Galaxy Multi Specialty Hospital
Project Overview
Create an OPD (Outpatient Department) management system for "Galaxy Multi Specialty Hospital" located at Sane Chowk, Nair Colony, More Vasti, Chikhali, Pimpri-Chinchwad, Maharashtra 411062.

Technology Stack:

Frontend: React.js with TypeScript
Styling: Tailwind CSS with Shadcn UI components
Backend: Node.js with Express
Database: PostgreSQL
Authentication: JWT-based (from external hms-core service)
Color Scheme & Theming
Primary Colors:
Primary (Medical Blue): HSL 197 71% 53% (Light Mode) / HSL 197 61% 63% (Dark Mode)
Background: HSL 0 0% 98% (Light) / HSL 215 28% 17% (Dark)
Foreground: HSL 215 28% 17% (Light) / HSL 0 0% 95% (Dark)
Status Colors:
Scheduled: Blue (bg-blue-100/blue-800)
Checked-in: Yellow (bg-yellow-100/yellow-800)
Completed: Green (bg-green-100/green-800)
Cancelled: Red (bg-red-100/red-800)
Design Tokens:
Border Radius: 0.5rem (8px) for cards, 0.375rem (6px) for buttons
Shadows: Subtle elevation using HSL-based shadows
Font: Roboto (sans-serif), Roboto Mono (monospace)
